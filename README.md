# Expense Manager

## Overview
Expense Manager is a desktop application developed using Java Swing and MySQL with JDBC connectivity. It allows users to record and manage daily expenses through an easy-to-use graphical user interface.

## Features
- Add Expense
- Update Expense
- Delete Expense
- View All Expenses
- Budget Tracking
- Total Expense Calculation
- Transaction Counter
- MySQL Database Connectivity

## Technologies Used
- Java
- Java Swing
- JDBC
- MySQL Workbench
- Eclipse IDE

## Database
Database Name: `expense_manager`

Table Name: `expenses`

## How to Run
1. Import the project into Eclipse.
2. Add the MySQL Connector JAR.
3. Create the database using the SQL script.
4. Update the MySQL username and password in `DBConnection.java`.
5. Run `ExpenseManager.java`.
